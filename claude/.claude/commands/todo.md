# Todo

Add a task to the project's TODO.md file for later follow-up.

## Usage

`/todo <task description>`

## Instructions

1. Append the task to `TODO.md` in the current working directory
2. Create the file if it doesn't exist, starting with a `# TODO` header
3. Format: `- [ ] <task>` (unchecked markdown checkbox)
4. Add a blank line before the new task if the file doesn't end with one
5. Confirm with a single short line (e.g., "Added: <task>")

Do NOT:
- Ask clarifying questions — just add what the user typed
- Add timestamps, categories, or metadata unless specified
- Read or summarize the existing TODO.md contents
- Interrupt the user's flow with lengthy responses
